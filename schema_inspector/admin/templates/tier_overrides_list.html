Как это влияет на runtime

После INSERT/UPDATE/DELETE override применится на следующий перезапуск live workers. Live workers загружают snapshot из таблицы один раз на boot (HybridApp.ensure_tier_override_registry) — поэтому после правки нужно sudo systemctl restart sofascore-live-tier-*@*.
